Nomenclatura y Números Cuánticos

Nomenclatura de compuestos químicos

Nomenclatura de compuestos inorgánicos

Compuestos iónicos

Regla General: El nombre del compuesto se forma combinando el nombre del catión (ión positivo) seguido del nombre del anión (ión negativo).

Ejmplo: NaCl se nombra como cloruro de sodio. El Na⁺ es el catión sodio y el Cl⁻ es el anión cloruro.

Compuestos Covalentes:

Regla General: Se usa prefijos para indicar el número de átomos de cada elemento y se termina con el nombre del segundo elemento con el sufijo "-uro".

Ejemplo: CO₂ se nombra como dióxido de carbono. "Di-" indica dos átomos de oxígeno.

Ácidos

Regla General: La nomenclatura depende de si el ácido es oxácido (contiene oxígeno) o hidrácido (no contiene oxígeno).

Ejemplo: HCl (en solución acuosa) se nombra como ácido clorhídrico (hidrácido). H₂SO₄ se nombra como ácido sulfúrico (oxácido).

Nomenclatura de compuestos orgánicos

Hidrocarburos

Regla General: Los nombres se basan en la cadena más larga de átomos de carbono y se utilizan prefijos para indicar la presencia de ramificaciones o grupos funcionales.

Ejemplo: C₆H₁₂ se nombra como hexano. La cadena de seis carbonos es la base y se le añade el sufijo "-ano".

Compuestos con grupos funcionales

Regla General: El nombre del compuesto se deriva del grupo funcional principal, con el sufijo apropiado.

Ejemplo: CH₃OH se nombra como metanol. El grupo funcional es un grupo hidroxilo (-OH).

Nomencaltura de iones e isótopos

Iones

Regla General: Los iones se nombran añadiendo el sufijo "-ión" para los aniones y simplemente mencionando el nombre del catión.

Ejemplo: Cl⁻ se nombra como ion cloruro y Na⁺ como ion sodio.

Isótopos

Regla General: Se utiliza el nombre del elemento seguido de su número másico.

Ejemplo: El isótopo del carbono con masa 14 se denomina carbono-14.

Números cuánticos

Tipos de números cuánticos

Número cuántico principal (N)

Descripción: Indica el nivel de energía del electrón y el tamaño de la órbita. Se puede considerar como la capa o nivel principal del electrón.

Valores: n = 1, 2, 3, 4, etc.

Ejemplo: En el átomo de hidrógeno, el electrón en el estado fundamental tiene n = 1.

Número cuántico secundario o azimutal (L)

Descripción: Indica el tipo de orbital y la forma del orbital. Está relacionado con el momento angular del electrón.

Valores: l = 0, 1, 2, ..., (n-1). Los valores corresponden a los tipos de orbitales: s (l=0), p (l=1), d (l=2), f (l=3).

Ejemplo: Para n = 2, l puede ser 0 (orbital s) o 1 (orbital p).

Número cuántico magnético (Mₗ)

Descripción: Indica la orientación del orbital en el espacio respecto a un campo magnético externo.

Valores: mₗ varía desde -l hasta +l, incluyendo cero.

Ejemplo: Para un orbital p (l = 1), mₗ puede ser -1, 0, o +1.

Número cuántico de espín (Mₛ)

Descripción: Describe el giro del electrón sobre su propio eje y está asociado con el momento magnético intrínseco del electrón.

Valores: mₛ = +1/2 o -1/2.

Ejmplo: Dos electrones en el mismo orbital tendrán espines opuestos, uno con mₛ = +1/2 y el otro con mₛ = -1/2.

Aplicación de los números cuánticos

Configuración electrónica: Los números cuánticos determinan la configuración electrónica de un átomo, que describe cómo los electrones están distribuidos en los orbitales.

Regla de aufbau: Los electrones llenan primero los orbitales de menor energía antes de ocupar los de mayor energía.

Principio de exclusión de Pauli: Ninguno de los dos electrones en un átomo pueden tener el mismo conjunto de cuatro números cuánticos.

Ejemplos y Aplicaciones

Ejemplo de configuración electrónica: El oxígeno (O) tiene 8 electrones, con configuración 1s² 2s² 2p⁴. Los números cuánticos para el electrón más externo (en el orbital 2p) son n = 2, l = 1, mₗ = -1, 0, o +1, y mₛ = +1/2 o -1/2.

Ejemplos de números cuánticos en un átomo: El átomo de nitrógeno (N) tiene una configuración 1s² 2s² 2p³. Los números cuánticos para los electrones en el orbital 2p serían n = 2, l = 1, mₗ = -1, 0, o +1, y mₛ = +1/2 o -1/2.
